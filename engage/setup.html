🔔 Notificaciones diarias

Configuración por única vez — después, la biblioteca te escribe sola.

1 · Token de GitHub (se guarda solo en este teléfono)

El token permite que este dispositivo guarde su suscripción y tus reacciones en el repo. Vive únicamente en el localStorage de este navegador; jamás se commitea.

  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Repository access: Only select repositories → Abecedeefege/Biblioteca
  3. Permissions → Repository permissions → Contents: Read and write. Nada más.

2 · Activar las notificaciones

Pide permiso al navegador, suscribe este dispositivo y guarda la suscripción en el repo para que el dispatcher sepa a dónde mandar.

3 · Probar de punta a punta

Encola un push con fecha en el pasado: el workflow se dispara al instante y en menos de un minuto debería sonar el teléfono, incluso con la app cerrada. Requiere el secret VAPID_PRIVATE_KEY ya cargado en Actions.
